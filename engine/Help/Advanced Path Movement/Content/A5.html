Connect Nodes

This action will create a connection from one node to another.

Say we had some nodes like this :

1                             3

                   2                                4

If you create a connection of node 1 to 3 this is what happens :

1 ---------------->  3

                   2                                4

So an object can move from node 1 to node 3. However, they will not be able to move back. If you want a 2 way connection, you will have to create another connection between node 3 and node 1. 

1 <-------------->  3

                   2                                4

When you make a connection, you will have to enter a speed. 1 is the default speed. 2 is twice as fast. 0.5 is half as fast. You could for instance create a connection of speed 2 from 1 to 3, and a connection of speed 0.5 from 3 to 1, then the object can travel faster if it goes to the right, and slower if it goes to the left. In a realistic situation, perhaps node 1 is uphill, and node 3 is downhill, so going from node 3 to 1 is easy as its downhill, whereas going from node 1 to 3 is a tiresome journey uphill.
