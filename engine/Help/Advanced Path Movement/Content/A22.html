Insert path node to journey

A journey is a set of instructions telling the player where to go next. Inserting the path node requires understanding the positions of the journey array.

0-Current point the object is standing on, or previous point if the object is between two points

1-The next point

2-The next point

 

The insert action inserts the item before the item. So say we have a path like this:

1  <--------------->  2
                                 /|\
                                  |
                                  |
                                  |
                                  |
                                 \|/ 
                                 3

  

We want the object to start at 1, move to 2, and then move to 3. Normally we would do it like this:

Add Node 1 to Journey
Add Node 2 to Journey
Add Node 3 to Journey

But say for some reason we HAD to do this

Add Node 1 to Journey
Add Node 3 to Journey

We need to insert a node to the journey between node 1 and 3. The array would currently look like this:

0 - Node 1
1 - Node 3

So to insert Node 2 into the journey, we insert it at 1. Then the array will look like this:

0 - Node 1
1 - Node 2
2 - Node 3

And the object will happy travel.
