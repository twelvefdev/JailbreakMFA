Path Maker

The Path Maker is a special little program that has a similar interface to the MMF path editor. The editor allows you to edit and create paths, and save them for use in the Advanced Path Movement Object.

 Toolbar

Selecting this tool allows you to create nodes by left clicking anywhere in the 'playfield'
Selecting this tool allows you to create nodes by left clicking anywhere in the playfield, while connecting the previous node to the new node allowing objects to move from the previous node to the new node but not travel back.
Selecting this tool allows you to create nodes by left clicking anywhere in the playfield, while connecting the previous node to the new node allowing objects to move from the previous node to the new node and back again.
Selecting this tool allows you to create a two way path of nodes by holding down the left mouse anywhere in the playfield and moving the mouse along the path the player can take. Object can walk both ways along these paths.
Selecting this tool allows you to create a one way path of nodes by holding down the left mouse anywhere in the playfield and moving the mouse along the path the player can take. Object can walk along these paths but only in one direction.
Selecting this tool allows you to destroy nodes. To destroy a node, left click on it and it will vanish and all connections to it will be 'broken'.
Selecting this tool allows you to connect two nodes. Left click the first node, then left click the second node, and the two will be connected. You can make 2 way connections by clicking the first, then the second, and then the second, then the first...but its faster to use the 2 way connect tool ( below )
Selecting this tool allows you to connect two nodes so the player can walk both ways through. Left click the first node, then left click the second node ( it doesn't matter which order ) and both nodes will be connected together.  
Selecting this tool allows you to break a connection between two nodes. Left click the first node, then left click the second node, and the connection of the first to the second will be destroyed.
Selecting this tool allows you to break a connection between two nodes. Left click the first node, then left click the second node. The connection of the first to the second, and the second to the first, will be destroyed.
Selecting this tool allows you to move nodes. Hold down the left mouse on a node, and drag it to its new position. Then let go the mouse. The node you moved will still be selected, you can use the arrow keys to move it by 1 pixel. To select more than 1 node, left click away from a node to create a selection box. Drag the box over the other nodes, then release. All the nodes within the box are selected. By left clicking on a node that is selected, all the selected nodes will be dragged. To unselect nodes, right click or left click away from them.


Whenever a connection is made between 2 nodes, a speed can be set. You must set this speed before you make the connection. 1 is the default speed. 2 is twice as fast. 0.5 is half as fast. You can either drag the slider, or you can enter a value in the editbox.
This will allow you to test the movement, just like MMF allows you to test the default movement.
Clicking this allows you to change the background.
Allows you to load a path. Select the file.
Allows you to save a path. You will need to enter the file name and its extension.

 

 Testing the movement

Right click near a node to bring up a popup menu. You will first have to select 'Position' so the 'car' knows where it starts. Then move the mouse to another node, and bring up the menu and select 'Find Path'. The car will determine the best way to get there. If the car doesn't start moving, then there is no possible path to the node.

Changing the background

Clicking this button will bring up a special wizard that will allow you to 'capture' a screenshot of your playfield. Follow the instructions. When finished, a slightly transparent screenshot will appear in your background and you can use it as a guide.
