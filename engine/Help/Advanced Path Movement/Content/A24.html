Delete path node in Journey

This will remove a node from the journey. Say we had a situation like this:

1  <--------------->  2  |\
                                 /|\   \
                                  |      \
                                  |        \| 
                                  |           4
                                  |        /|
                                  |      /
                                \|/    /
                                 3  |/

And the journey array was created with 4 actions

Add node 1 to journey
Add node 2 to journey
Add node 3 to journey
Add node 4 to journey

However, this path is kinda silly, because it would be much faster to move from 1 to 2 to 4 rather than take the long way round and go to 3. The modify this path all we would have to do is remove Node 3 from the journey. This is what the journey array instructions look like:

0 - Node 1
1 - Node 2
2 - Node 3
3 - Node 4

So we need to remove Node 3, and because its position is 2, we remove Path node ID 2.

 

0-Current point the object is standing on, or previous point if the object is between two points

1-The next point

2-The next point

 

Say we had to instructions to go to node 1, then 3, then 4, then 9. This action would remove node 9.
