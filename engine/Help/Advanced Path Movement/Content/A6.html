Disconnect Nodes

This action will disconnect a connection from one node to another.

Say we had some nodes like this :

1 ---------------->   3

                   2  <-------------------->   4

1 connected to 3, 2 connected to 4, and 4 connected to 2.

If we disconnect the connection 1 to 3, we get this:

1                             3

                   2  <-------------------->   4
