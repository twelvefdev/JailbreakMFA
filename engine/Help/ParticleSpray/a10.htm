Set Transparency Coefficient

If your particle objects have the Semi-transparency option on this value will affect it's transparency. It controls how fast the object fades. 1 will make it change linear. If the value is close to 0 it will fade out pretty fast, the greater the value is the longer it will wait before it fades the objects.
Set this value to 0 if you don't want the extension to handle automatic transparency changing.
Negative values will give you some flashy effects :)

 

Parameters:

TypeDescription
ValueSprayer ID
ValueTransparency Coefficient (1=linear)
