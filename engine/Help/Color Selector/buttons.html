Additional Button Conditions:

Top/Bottom Button Hit (Triggered Conditions)
These conditions are triggered when the respective button is clicked in the color popup, AFTER any default actions are performed (such as setting the current color or opening a color dialog).

Additional Button Actions:

Top Button > Set Function > Nothing
Makes the top button have no specific function.

Top Button > Set Function > Set Color
Makes the top button automatically set the current color.

Top Button > Set Color > None
If the top button sets the current color, this makes it set the current color to none.

Top Button > Set Color > Nothing
If the top button sets the current color, this allows you to choose a custom "default color" for it to use.

Top Button > Display > Show Color Box
If the top button sets the current color, this allows it show a color box on the left that has that color (unless it sets the current color to none).

Top Button > Display > Do Not Show Color Box
The top button will only display text.

Top Button > Display > Set Text
Sets the text displayed for the top button. Set the text to an empty string ("") to eliminate the top button.

Bottom Button > Set Function > Nothing
Makes the bottom button have no specific function.

Bottom Button > Set Function > Open Color Dialog
Makes the bottom button automatically open the Windows Color Dialog.

Bottom Button > Set Text
Sets the text displayed for the bottom button. Set the text to an empty string ("") to eliminate the bottom button.
