Color Dialog Conditions:

Closed by OK (Triggered Condition)
This condition is triggered whenever the user ends the color dialog by pressing OK.

Closed by Cancel (Triggered Condition)
This condition is triggered whenever the user ends the color dialog by pressing Cancel.

Color Dialog Actions:

Open Color Dialog
This action manually opens the Windows Color Dialog, using the Color Selector's current color as the default color.

Set Custom Color
Sets one of the 16 custom colors for the Windows Color Dialog. The first parameter is the 0-based custom color index (0 to 15) and the second parameter is the actual color.

Set Display Information
Sets how the Windows Color Dialog is displayed. Add together the flag expressions you want for the parameter in this action.

Color Dialog Expressions:

Display Info Flags
These are self-explanatory, just add them together for the Set Display Information action.

Get Custom Color
Returns one of the 16 custom colors for the Windows Color Dialog. The parameter is the 0-based custom color index (0 to 15).
