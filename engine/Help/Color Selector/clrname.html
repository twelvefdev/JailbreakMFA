Color Name Actions:

Set/Add Color Name
Sets the name used for the specified color.

Remove Color Name
Removes the name used (if any) for the specified color.

Save/Load Color Name Settings
Saves or loads the color name settings to or from a file.

Load Default Color Name Settings
Restores the color name settings to their initial state at the start of the frame. See the bottom of this page for the initial color name definitions.

Color Name Expressions:

Get Color from Name
Returns the color value that has the specified name definition, or -1 if that definition does not exist.

Get Name from Color
Returns the name that the specified color is defined as, or an empty string ("") if that definition does not exist.
