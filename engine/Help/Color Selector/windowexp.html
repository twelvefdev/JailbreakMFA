Window Expressions:

Get X Position
Returns the current X position of the color button.

Get Y Position
Returns the current Y position of the color button.

Get Width
Returns the current width of the color button.

Get Height
Returns the current height of the color button.

Get Button Window Handle
Returns the handle to the color button window. It's unlikely that you'll ever need to use this.

Get Popup Window Handle
Returns the handle to the color popup window, if it exists. Otherwise, it returns 0. It's unlikely that you'll ever need to use this.
