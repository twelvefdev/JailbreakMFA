Color Selector Documentation

What is Color Selector?
Color Selector is an extension which creates a button or combo box that, if clicked, displays a popup menu that provides a simple way of selecting a color. It also provides access to the Windows color dialog.

Setup Box



When you insert a Color Selector, you will be prompted with this dialog of options. Here we will go through what each one means.

Button: The color selector appears as a button.


Combo Box: The color selector appears as a combo box.


Tracking Selection: When this option is on, the selection will be changed to whatever color the mouse is over while the color popup is still open. Normally, the selection will only change once the color popup closes.

Combo Box Doesn't Display Text: Instead a large color rectangle is shown.


Default Color: This sets the color the top button uses as a "default color."

Use XP Theming: Unfortunately, there is no way for this extension to know whether you've enabled XP Visual Themes in your application runtime settings. For proper display results, you should keep this option the same as your application runtime setting.

Top Button Sets Color: By default, the top button doesn't do anything. But, if this option is checked, the top button will automatically set the color for you.

Top Button Sets Color to None: The top button sets the color to a special "color," none. Otherwise the top button will use the Default Color.

Top Button Shows Color Box: If this is checked, the top button will not only display text, but a box with the default color too (unless the top button sets the color to none).

Top Button Text: Self-explanatory. If you don't want a top button, simply erase all of this text.

Bottom Button Sets Color: By default, the bottom button doesn't do anything. But, if this option is checked, the bottom button will automatically pop up the Windows color dialog when clicked.

Bottom Button Text: Self-explanatory. If you don't want a bottom button, simply erase all of this text.
