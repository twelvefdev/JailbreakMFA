Miscellaneous Conditions:

Color Selection Changed (Triggered Condition)
This condition is triggered whenever selected color changes (which can happen while the color popup is open if tracking selection is on. Also, if the current color is the "default color," and the default color is changed through an action, this condition is triggered.

Color Picker Dropdown (Triggered Condition)
This condition is triggered whenever the user opens the color popup, by either an action or clicking the color button.

Color Picker Closeup (Triggered Condition)
This condition is triggered whenever the user closes the color popup, by either an action, selecting a color, or clicking outside the popup.

Is Popup Visible?
Returns true if the color popup is currently visible.

Miscellaneous Actions:

Show/Hide Color Popup
Manually show or hide the color popup.

Miscellaneous Expressions:

Get Combo Box Text
Get the text the combo box currently is or would display, according to the rules you have set in the combo box options.
