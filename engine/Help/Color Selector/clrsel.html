Color Name Conditions:

Is Color Default?
Returns true if the selected color is the default color. To be the default color, the top button must have been clicked, not a button that has the same color as the default color.

Is Color None?
Returns true if the selected color is none.

Color Name Actions:

Set Selected Color
Sets the current selected color. -2 is the default color, -1 is no color.

Set Selected Color to None
Sets the current selected color to none.

Color Name Expressions:

Get Selected Color
Returns the selected color. If the color is none, this value will be -1. If the color is default, it will return the actual default color.
