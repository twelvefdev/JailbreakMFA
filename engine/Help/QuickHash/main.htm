Quick Hash Documentation

What is Quick Hash?
Quick Hash is an extension which can hash a file using many different algorithms.

What algorithms are supported?
Currently 10 hashing algorithms are supported:

- GOSTHash
- G-Hash
- MD2
- MD4
- MD5
- SHA-1
- SHA-2 256 bit
- SHA-2 384 bit
- SHA-2 512 bit
- Tiger

What are the settings on the setup box?
These control how the hash is outputted, and are as follows:

A hash of "Hello world" using SHA-1:

Lowercase, no spaces:
7b502c3a1f48c8609ae212cdfb639dee39673f5e

Lowercase, spaces:
7b 50 2c 3a 1f 48 c8 60 9a e2 12 cd fb 63 9d ee 39 67 3f 5e

Uppercase, no spaces:
7B502C3A1F48C8609AE212CDFB639DEE39673F5E

Uppercase, spaces:
7B 50 2C 3A 1F 48 C8 60 9A E2 12 CD FB 63 9D EE 39 67 3F 5E

Note: Formatting is NOT applied to a GHash because of it's length.
