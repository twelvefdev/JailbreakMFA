MD algorithms:

MD2:
Hashes with MD2 algorithm. (32 characters, 128bit)
Warning, this algorithm is pretty slow as it is old, and is not recommended generally.

MD4:
Hashes with MD4 algorithm. (32 characters, 128bit)
Used by many file sharing applications, still pretty secure, recommended for general use.

MD5:
Hashes with MD5 algorithm. (32 characters, 128bit)
Still pretty secure, used by several well known security applications.
