Delete vertical line

This action takes an X coordinate. It then deletes all bricks in the certical line where the X coordinate is equal the line of your choice. The next parameter takes a value. If you leave it as 0, the line will simply be deleted and nothing more will happen. If you enter '1' as the value, all bricks above the deleted line, will be moved one slot downwards. If you enter '2', all lines below the deleted line are moved one slot upwards. All bricks moved will trigger a "On brick moved" condition and all deleted bricks in the line will trigger a "On brick deleted" condition.
