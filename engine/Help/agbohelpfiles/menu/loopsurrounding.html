Loop surrounding

This action will loop through all bricks around a given coordinate. In total it will trigger "On looped brick" 8 times unless the given brick is in one of the edges, then its of course lesser bricks looped.
If i give AGBO the position of the red square, it will loop all the blue bricks. Could be used in minesweeper games for example.

Parameters:
X
Y

When you start a loop, its like starting a fastloop. AGBO will trigger a condition similar to an "on loop" condition for each brick that you told AGBO to loop through.
