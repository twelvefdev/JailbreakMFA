Loop all bricks of type

This action is similar to the "Loop all bricks in board" but it will only trigger "On looped brick" when it finds a brick of the given type.

Parameters:
Bricktype - the type of brick AGBO will look for in the board

When you start a loop, its like starting a fastloop. AGBO will trigger a condition similar to an "on loop" condition for each brick that you told AGBO to loop through.
