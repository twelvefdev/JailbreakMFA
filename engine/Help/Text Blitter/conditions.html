On Begin

Is triggered when the object begins drawing the text

On End

Is triggered when the object ends drawing the text

On Line Begin

Is triggered when the object begins drawing a line of text

On Line End

Is triggered when the object ends drawing a line of text

On Character

Is triggered when the object is about to draw the next character

Transparent

The transparency of the object alters how Multimedia Fusion draws the object in the frame. When transparency is turned on, the transparent colours selected on the image show as transparent and can be seen through. However, when transparency is turned off, the transparent areas appear as black on the screen.

This condition will allow the actions associated with it to be executed when the transparent option within the extension is turned on

Text Functions

This condition tests whether the text function option is enabled within the extension

Word Wrapped

This excellent new feature standard to the extension is known as Word Wrapping and takes the strain away from making sure long pieces of text appear on the screen correctly. As standard, when Word Wrapping is enabled any words that may be cut-off by the edge of the extension's boundary will be wrapped to the next line

This condition will allow the actions associated with it to be executed when the word wrapping option within the extension is turned on

Multiline

Sometimes you may wish to prevent the object from moving onto multiple lines, you can do it from these set of actions. When multiline is enabled the extension freely creates multiple lines for both the correct characters and when word wrapping (word wrapping does not work when multiline is disabled)

This condition will allow the actions associated with it to be executed when the multiline option within the extension is turned on
