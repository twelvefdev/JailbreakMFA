Examples

Welcome to the examples section, below are references to the examples that come with text blitter


Original examples

01 - Animation Example - basic

Not sure what parameters to use for the 'Cos Scroller' animation, well, this simple example allows you to play around with the settings at runtime and see the result live. As a task, you could convert the example into a full featured animation previewer that allows you to select the different animations and alter the parameters of each.

02 - Character Map Example - basic

In the event that a bitmap doesn't follow a standard order, this example shows you how to set up the extension in a way that allows you to use it.

This example does not include any events

03 - Scrolling Credits - basic

Again, another very basic example showing you how easy it is to put together a simple demo without even using any events.

The scrolling is handled by the object using the parameters defined in the setup dialog. The example also demonstrates the vertical fade effect on marquee animations

04 - Automatic Scrolling - Intermediate

Another friarly simple example shows how you can create an automatic scrolling engine, maybe for an RPG style text box. (Check further down for a true typing engine). The example allows you to type text into the exit box which is then copied to the Text Blitter, through a couple of events the extension will scroll until the last line of text is most visible.

05 - Flag Example - basic

Again, no events here, all the work is handled by the extension. This example shows you how to construct a simple waving flag using the 'Sin Wave' animation. The rest of the example is just backdrop objects which make the effect appear more real.

06 - Animation Emulator - Intermediate

Using different options this time (see the other Animation example below) this example provides the same effect. The effect is the same, however, in my opinion is easier to create

07 - Scrolling Credits - basic

Another scrolling credits example with a different bitmap this time. This example doesn't include any events

08 - Scrolling RPG Text - Intermediate

Something on the borderline of advanced, this example shows you how to create a simple scrolling typing engine. The vertical fade effect ensures the text is smoothly created and moved off the screen. This is a never before seen effect in multimedia fusion!!

09 - Animation Emulator - Advanced

This example includes two frames, the first shows how to do a simple animation effect cycling through different pictures on a bitmap. The second frame plays the same animation forwards and backwards in a repetitive loop

10 - Active Object Emulator - Advanced

Although the principle of this example is fairly simple, its quite an advanced technique and something which you will find extremely interesting. The example shows how to emulate an active object with animations and directions. The image could also be stored externally allowing users to customise your game

Balloon examples

11 - Balloons - Intermediate

Using a simple script engine, this example uses the balloon effects within the text blitter to create a speach-engine between two players.

Callback examples

12 - Callback - Trails - Intermediate

Using the list controls, this example creates a simple mouse trail effect

13 - Callback - Rotating - Intermediate

In this example, the text rotates around

14 - Callback - Blitting - Advanced

A simple way of blitting different sized 'sprites' to the screen.

15 - Callback - Alternate Lines - Intermediate

The callback routine changes the image offset every even line. There are 2 character-maps stored on the internal image, one lighter than the other.

16 - Callback - Real-time-lighting - Advanced

Using the transparency options with the callback routines, this example shows you how you can create a nice looking real-time-lighting engine over the text
