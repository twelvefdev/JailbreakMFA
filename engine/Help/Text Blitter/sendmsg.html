Send Message

This special action allows you to send messages to the extension, and edit specific attributes that are otherwise not available in the rest of the extension.

This action is designed for advanced users and those with programming experience, so dont be put-off if you dont undertand any areas of this page, the rest of the extension is designed to be as easy to use as possible

If you are a newbie to Text Blitter or even to mmf, Steer-clear of this section! :P

Messages

The messages below are given in decimal values, and can be directly entered into the "message" argument of this action. When you send a message to the extension, you can also sent it two 'long' arguments.


Object Properties
16 - Set Object Transparency ( transparency, 0 )
17 - Set Object Flags ( flags, 0 )

Callback
32 - Set Callback flags ( flags, 0 )

Balloons
48 - Set Balloon Flags ( flags, 0 )
49 - Set Balloon Position ( x, y )
50 - Set Balloon X Position ( x, 0 )
51 - Set Balloon Y Position ( y, 0 )
52 - Set Balloon Handle ( handle, 0 )
53 - Set Balloon Source ( x, y )
54 - Set Balloon X Source ( x, 0 )
55 - Set Balloon Y Source ( y, 0 )

56 - Set Balloon Left Padding ( value, 0 )
57 - Set Balloon Top Padding ( value, 0 )
58 - Set Balloon Right Padding ( value, 0 )
59 - Set Balloon Bottom Padding ( value, 0 )

16 - Set Object Transparency

This action allows you to set directly the transparency of the extension

1: Transparency value (0-128)
2: Unused (zero)

17 - Set Object Flags

This action allows you to DIRECTLY alter the internal flags of the extension. You must use the binary arithmetic functions in MMF if you wish to manipulate these values correctly

0x00000001 - Extension uses transparent color
0x00000002 - Extension uses text functions
0x00000020 - Extension uses word wrapping
0x00000040 - Extension allows multiple lines
0x00000400 - Edge fading is used on the maquee animation

Some of the options are cached throughout the update process and the effect may not become apprant until the next MMF-cycle.

1: New Flag value
2: Unused (zero)

32 - Set Callback flags

Here you can turn which callbacks are to be called. This action will ONLY change he callback flags within the extension, unlike the 'Set Object Flags' function, which alters everything.

0x00000800 - Callback system enabled
0x00001000 - OnBegin callback enabled
0x00002000 - OnEnd callback enabled
0x00004000 - OnBeginLine callback enabled
0x00008000 - OnEndLine callback enabled
0x00010000 - OnCharacter callback enabled

The extension uses a small cache through the drawing process to help speed things up, so your changed wont take effect until the next mmf cycle.

1: New Flag value
2: Unused (zero)

48 - Set Balloon Flags

Here you can alter the internal-flags for the balloon properties. The action allows you to only change this flags below.

0x00020000 - Enables the baloon
0x00040000 - Inverts the handles
0x00080000 - Keep on scren

The extension uses a small cache through the drawing process to help speed things up, so your changed wont take effect until the next mmf cycle.

1: New Flag value
2: Unused (zero)
