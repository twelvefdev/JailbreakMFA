Set offset multiplier

The offset multipliers can be used to do various effects on lens. It's best to experiment with the values and see what they do.

Some observations:
Negative values flips the image horizontally or vertically, large values makes the result look "compressed" while small values will make the result look more zoomed in. (you can that way control a zoom value both horizontally and vertically)

Parameters:
X or Y multiplier. (float value)
