 Information


This object allows you to retrieve and change the position of the mouse cursor. It can also retrieve how far the mouse has rolled. This can mimic the speed of the mouse.

Click and Create and Multimedia Fusion already can retrieve the mouse position. However, they retrieve the position relative to the top-left corner of the frame.

This object retrieves the position relative to the top-left corner of the screen.

Conditions

This section explains all of the conditions you can use to check various things with the object. You create a condition from the event editor by double clicking the "New Condition" at the bottom of the chart. Then right click on the object and select a condition.

Compare Actual Mouse X Use this Condition to compare the horizontal position of the mouse relative to the left side of the screen. This is measured in pixels.
Compare Actual MouseY This compares the vertical mouse position relative to the top of the screen. This is also measured in pixels.
Compare Distance rolled This compares how far the mouse has rolled since the last loop. If this equals zero, then the mouse has not moved. This distance is measured in pixels.

 Actions

This section explains all of the actions you can use to perform actions with this object. You create an action by right clicking in the appropriate column corresponding to the object, and selecting an action from the popup menu.

Set Actual Mouse X This action changes the horizontal position of the mouse relative to the left of the screen.
Set Actual Mouse Y This changes the vertical position of the mouse relative to the top of the screen.

 Expressions

This section explain what all of the expressions in this object do. You get expressions by clicking Retrieve data from an object button in the expression editor. Then right clicking the object in the dialog and selecting an expression. This will insert a mini code into your expression.

Get Actual Mouse X Position This expression will retrieve the horizontal position of the mouse relative to the left of the screen. This is in the same format as the actions and condition, so if you set the mouse x position to this expression, it will stay the same.
Get Actual Mouse Y Position This expression will retrieve the vertical position of the mouse relative to the top of the screen.
Get Distance Rolled This expression will return the number of pixels the mouse rolled since the last loop.

Tips

You can set the mouse position, but if you set the mouse position to the x and y of an active object it doesn't put it in the correct place.

If you look at the picture to the right, it helps illutrate the problem. The X and Y you get from the active object refer to the top left corner of the game's frame. (The red circle.)

When you are setting and retrieving the mouse position, it is relative to the screen. (The blue circle.) You can find out the offset difference by subtracting the mouse position relative to the frame (XMouse and YMouse) from the Actual X and Y Mouse, which this object returns. This will retrieve the difference between the frame X and Y and the screen's X and Y.



You can use the expression that retrieves the distance the mouse rolled to get the speed the mouse is rolling. If this equals zero, then the mouse is not rolling, if this is 200, then the mouse is moving very fast.
You should realize that the mouse has a speed limit based on the user's mouse settings.
