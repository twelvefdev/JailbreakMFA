Actions


Change Mode To...


Colorize


Colorize mode is the most advance more yet the mode that colorizer do the fastest.
The Colorize mode takes 9 Parameters:
New red out of Old red (RR)
New red out of Old green (RG)
New red out of Old blue (RB)
New green out of Old red (GR)
New green out of Old green (GG)
New green out of Old blue (GB)
New blue out of Old red (BR)
New blue out of Old green (BG)
New blue out of Old blue (BB)
(The 2 letters between the brackets are the name used in the code explaination)
The color is calculated by this code: 

NewRed=Red*RR+Green*RG+Blue*RB;
NewGreen=Red*GR+Green*GG+Blue*GB;
NewBlue=Red*BR+Green*BG+Blue*BB;

Hue


This is a less advanced mode, yet slower to calculate because of the complex RGB to HSL and HSL to RGB functions.
The calculation is based on HSL.
The Hue mode takes 1 Parameter:
Hue (H)
(The letter between the brackets are the name used in the code explaination)
The color is calculated by this code: 

//Hue, Sat and Lum was converted from a RGB color.
NewHue=(Hue+H)%360;
NewSat=Sat; //No Change
NewLum=Lum; //No Change
//NewHue, NewSat and NewLum are now converted back to RGB.

HSL


HSL Mode is more advanced than Hue mode, which changes only Hue, yet slower to calculate than Colorize mode because of the complex RGB to HSL and HSL to RGB functions.
The calculation is based on HSL.
The Hue mode takes 3 Parameter:
Hue (H)
Sat (S)
Lum (L)
(The letter between the brackets are the name used in the code explaination)
The color is calculated by this code: 

//Hue, Sat and Lum was converted from a RGB color.
NewHue=(Hue+H)%360;
NewSat=Sat+S;
NewLum=Lum+L;
//NewHue, NewSat and NewLum are now converted back to RGB.

Options


Colorize mode


This Submenu include 9 actions:

Set New red out of Old red
Set New red out of Old green
Set New red out of Old blue
Set New green out of Old red
Set New green out of Old green
Set New green out of Old blue
Set New blue out of Old red
Set New blue out of Old green
Set New blue out of Old blue
They all set the 9 parameters for the Colorize Mode.

Set value for Hue mode


This action sets the H parameter used by Hue mode and HSL mode.

Set values for HSL mode


This action sets the H, S and L parameters used by HSL mode. (The Hue parameter effects both Hue and HSL mode.)

Fading


This Submenu includes actions to fade all parameters used in all 3 modes from one to another in a selected length in frames. (If the FPS is set to 50 (MMF2's Default), then 1 second=50 frames.)

Resize colorizer


Changes the size of colorizer.
If masking is used the mask will be temporary resized, no data will be lost.

Load and Save a preset file


Those 2 actions save/load the selected mode and parameters for all 3 modes to/from a 52 bytes sized file.

Load mask from Overlay


If you're using a mask (Checked the Mask option in the properties toobar) then you are able to load images from Overlay to make dynamic masks.
You only have to enter the Overlay's fixed value whenever it updates
You can change the first image by clicking the "..." button next to the mask option in the properties toobal.

Update Colorizer in Manual Update mode


If yo don't want to use MMF2's and Colorizer's automated updating, you can check the "Use Manual Update" option in the properties toolbar.
Colorizer will update only when the action is called.
If you do not understand how MMF2's graphics engine work, It is Not recommended you will use this option.

Change Threshold value for Masking


If you're using a mask (Checked the Mask option in the properties toobar) then you are able to change the threshold value to make dynamic masking using 1 image. The Masking engine uses threshold value like this:

if(MaskRedPixel>Threshold) //Note - Only the red channel is in use! Not a greyscale of all 3 channels.
{
	NewRed=...;
	NewGreen=...;
	NewGreen=...;
	// The ...s are the calculations explained in "Change Mode To...".
{
else
{
	NewRed=Red;
	NewGreen=Green;
	NewBlue=Blue;
}

Expressions

Get Width/Height


Returns the Width/Height of Colorizer.

Get Red of Red, Blue of Blue, H, S and L etc.


Those return the parameters used with Colorizer, Explain in "Change Mode To...".
