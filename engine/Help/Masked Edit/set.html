Edit Mask Actions:

Set Mask
Sets the rules for the edit window and erases any current data.

Set Data
Sets the data for the edit window. The data is received in one long string. For the mask 

[a-z][a-z][a-z] [0-9] {0-99}
you should input data in the following format: abc509
For number cells, you should always use the number of digits that the highest possible number has, otherwise your data won't be parsed properly.
For a non-restrictive edit, this action just sets the text of the edit.

Set Prompt Character
Sets the character displayed when a cell has no data. By default this is the underscore (_).

Set Cell (Character)
Sets the character currently in the specified cell.
First expression: 0-based cell
Second expression: character to use
This action does nothing for a non-restrictive edit.

Set Cell (Number)
Sets the number currently in the specified cell.
First expression: 0-based cell
Second expression: number to use
This action does nothing for a non-restrictive edit.
