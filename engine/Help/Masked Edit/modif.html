Modification Conditions:

On Successful Modification (Triggered Condition)
This condition is triggered whenever the user makes a successful modification to the Edit. This includes both typing in a character or number cell or scrolling a number cell.

On Unsuccessful Modification (Triggered Condition)
This condition is triggered whenever the user makes an unsuccessful modification to the Edit. This includes both typing in a character or number cell or scrolling a number cell to a limit when number wrapping is off.
