Window Actions:

Set X Position
Sets the current X position of the Edit window.

Set Y Position
Sets the current Y position of the Edit window.

Set Width
Sets the current width of the Edit window.

Set Height
Sets the current height of the Edit window.

Enable
Enables the Edit window.

Disable
Disables the Edit window.

Show
Shows the Edit window.

Hide
Hides the Edit window.

Set Focus On
Sets keyboard focus to the Edit window.

Set Focus Off
Removes keyboard focus from the Edit window.
