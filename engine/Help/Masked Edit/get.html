Edit Mask Expressions:

Get Data
Gets only the editable characters/numbers from the edit mask and returns them as a string.
This expression returns an empty string for a non-restrictive edit.

Get Text
Returns the exact text displayed in the edit.

Get Character at Cell
Returns the character from the 0-based cell specified. If the cell is a number cell, it returns the number as a string. If the cell currently has no data, it returns an empty string.
This expression returns an empty string for a non-restrictive edit.

Get Number at Cell
Returns the number from the 0-based cell specified. If the cell is a character cell, it returns 0. If the cell currently has no data, it returns 0.
This expression returns 0 for a non-restrictive edit.

Get Token (String)
Returns the text from the 0-based token specified. A token is a group of editable characters. Each token is separated by static characters. If a cell in the token currently has no data, there is a prompt character in its place in the string returned.
This expression returns an empty string for a non-restrictive edit.
