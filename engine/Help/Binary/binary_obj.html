The Binary Object is a powerful new extension object which allows you to handle, filter, save, load, modify, search/replace, etc. binary data. You can set and get strings, binary data chunks stored in banks, short integers, long integers, and even single bytes using simple offset based functions.

Using the binary object you can do several types of filtering including:
  • Binary to ASCII Encoding using Base64, Hex encoding, or Quoted-Printable

  • Encryption using Blowfish

  • Compression using Zlib

  • Packing, or taking several files and packaging them into one file

Features
  • Save a binary directly to file either as a new file or appending to an existing file.

  • Load a binary from any file and append to, insert into, or replace some or all of the existing binary.

  • Swap binaries to a memory bank where you can handle multiple binaries at once using an associated key for access.

  • Ability to crop down or remove bytes from a binary.

  • Extensive find/replace functions, 25 complete replace functions from replacing a bank with another bank, to replacing a short integer with a string.

  • Count the number of occurrences of a specific byte, bank, string, short, long, etc.

  • Binary to ASCII encoding algorithms include Base64 and Quoted-Printable which will allow you to convert binary to ASCII (ie: to be used as SMTP email attachments with MooSock)

  • Blowfish Encryption, with a create random key function to produce encryption level random keys

  • Compression with the zlib algorithm (used in ZIP, PNG, etc.)

  • Bank keys can be compared with or without case sensitivity, depending on your preference

  • Banks can be directly saved to and loaded from files without touching the main binary

  • Function to retrieve an MD5 Signature for a binary

  • Save and load workspaces which allow you to save/load/merge the main binary and all of its banks to and from a single file

  • Get the memory addresses for a binary of any bank which can be used as arguments to CallDLL (to pass structures for example)

  • New feature: Encode to Hex and Decode from Hex


    • v1.03 to v1.04
    Fix: A bug relating to Base64 Encode and Decode has been fixed from versions 1.03 to 1.04
    Note: Quoted-Printable will only work on ASCII data, not binary data, it can be used to excape non-printable characters
    New: Encode to Hex and Decode from Hex have been added.
