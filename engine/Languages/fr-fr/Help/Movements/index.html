Introduction

Les mouvements sont des propriétés qui peuvent être attachées à beaucoup d'objets visuels dans CF 2.5, principalement aux objets actifs. Comme leur nom l'indique, les mouvements donnés à un objet permettent de le déplacer sur l'écran. Les mouvements sont un moyen simple pour vous de créer des effets complexes sans avoir à écrire un moteur de déplacement complet pour chaque objet. Chaque mouvement a des propriétés personnalisables vous permettant de l'adapter à vos besoins. La plupart du temps, il suffit de régler quelques valeurs dans le panneau de propriétés de l'objet, mais vous pouvez aussi vouloir changer ces valeurs en runtime via l'éditeur d'événements. Un certain nombre de mouvements génériques sont fournis avec CF 2.5 et des mouvements additionnels peuvent être ajoutés.

Les mouvements additionnels sont une nouveauté importante dans CF 2.5, il est maintenant possible d'ajouter des mouvements comme on ajoute des extensions. Ces mouvements peuvent être développés par des utilisateurs grâce au SDK de CF 2.5, une connaissance sommaire du langage C++ ainsi que le compilateur approprié.

Les versions précédentes de la gamme de produits Clickteam contenaient uniquement des mouvements prédéfinis (statique, balle rebondissante, chemin enregistré, souris, 8-directions, voiture et plateforme). Ces mouvements sont documentés dans l'aide de CF 2.5, légèrement différemment des mouvements additionnels. Pour introduire le nouveau concept de mouvements additionnels, Clickteam a créé un nouveau mouvement: Flipper. Le source code de ce mouvement est fourni dans le SDK pour ceux qui veulent voir comment fonctionnent les mouvements, ou même l'améliorer.

Une autre nouveauté dans CF 2.5 est que les objets peuvent maintenant avoir plusieurs mouvements. L'éditeur d'événements vous permet de changer de mouvement en cours d'exécution. Les versions précédentes des produits Clickteam n'autorisaient qu'un seul mouvement, obligeant par là même à créer plusieurs objets identiques avec des mouvements différents pour palier à ce problème.

Installation manuelle des mouvements additionnels

Dans la plupart des cas, le développeur fournit un programme d'installation installant l'extension mouvement aux bons endroits. Dans le cas contraire, donc si vous désirez installer l'extension mouvement à la main, il suffit de copier les fichiers .mvx des mouvements dans le répertoire Extensions et dans le répertoire Data\Runtime de CF 2.5. Note: si le développeur fournit des fichiers .mvx différents pour le répertoire Data\Runtime, vous devez installer ceux-là dans ce répertoire. Vous devez redémarrer CF 2.5 pour que les nouveaux mouvements soient pris en compte.

Utilisation des mouvements

Une fois que le mouvement a été installé, lancez CF 2.5 et démarrez un nouveau projet. Insérez un objet comme par exemple un objet actif dans l'éditeur de scènes, puis cliquez sur l'onglet Mouvements dans la fenêtre de propriétés. Cliquez sur la propriété "Type" pour afficher la liste des mouvements disponibles, comme indiqué dans la figure 1.

Figure 1: Liste des mouvements, votre propre liste peut être différente, elle dépend des mouvements installés.

Lorsque vous sélectionnez un mouvement dans la liste, les propriétés de ce mouvement seront affichées. Vous pouvez ensuite les configurer. Veuillez vous référer à la documentation du mouvement pour plus d'infos à propos de ses propriétés.

Extensions Contrôleur de Mouvements

Malheureusement, pour le moment un mouvement ne peut pas interagir directement avec le menu Mouvements dans l'éditeur d'événements pour afficher les actions, conditions et expressions qui lui sont propres. Cependant pour contourner le problème il est possible d'utiliser une extension pour "contrôler le mouvement". Cet objet d'extension doit être ajouté dans la scène et vous permet, par l'intermédiaire de ses propres menus dans l'éditeur d'événements, de récupérer ou de modifier les propriétés des mouvements pendant l'exécution de l'application.

Il est recommandé de télécharger le SDK de CF 2.5 qui inclut le "Movement SDK" et le code source du mouvement Flipper (Pinball) ainsi qu'un fichier d'aide.
