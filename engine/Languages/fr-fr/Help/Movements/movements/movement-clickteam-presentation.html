Mouvement présentation

Description

Ce mouvement est conçu pour aider à la création de présentations ou de diaporamas avec CF 2.5. Pour de simples présentations, aucun évènement n'est nécéssaire, toute la configuration peut être réalisée dans l'éditeur de scène et le mouvement se charge de la gestion, jusqu'au changement de scène! Inutile de positionner les objets en dehors de la scène pour les voir arriver dans l'affichage : positionnez-les comme il doivent être au final, et l'objet se charge de les faire apparaître.

Utilisation du mouvement

Tout objet avec ce mouvement peut avoir une configuration d'entrée et/ou de sortie (une de chaque). Ceci détermine quand l'entrée ou la sortie est déclenchée. Les options inclues sont aucune, apparition ou entrée/sortie à partir de plusieurs directions. Les objets avec une entrée ou une sortie afficheront une propriété de vitesse supplémentaire. Tous les objets avec une configuration autre que "aucune" demandent une valeur d'index. Celle ci indique l'ordre d'apparition/disparition, plusieurs objets peuvent partager le même index pour les faire apparaître/disparaître en même temps. Une valeur de zéro indique que le mouvement se produit au début de la scène.

Pour déclencher les mouvements en fonction de l'appui sur une touche par l'utilisateur, comme dans de nombreux logiciels de présentation, toutes les propriétés globales doivent être cochées. Voyez plus bas une description de ces propriétés; en résumé elles indiquent au mouvement de détecter les appuis sur une touche et même de changer la scène lorsque c'est nécessaire. Les utilisateurs plus exigeants peuvent utiliser l'objet de contrôle du mouvement.

Propriétés

Figure 1: propriétés du mouvement présentation.

Propriétés d'entrée

Type: défini le type d'entrée.
Vitesse: la vitesse pour les entrées en mouvement.
Index: défini quand l'entrée aura lieu. Un zéro indique quand la scène débute.


Propriétés de sortie

Type: défini le type de sortie.
Vitesse: la vitesse de sortie, pour les sorties en mouvement.
Index: défini quand la sortie aura lieu. Un zéro indique quand la scène débute.


Propriétés globales

Ces propriétés sont partagées par tous les objets avec un mouvement présentation dans la scène. Changer une valeur dans l'un des objets affecte tous les autres objets avec ce mouvement. A contraire de nombreux mouvements, le mouvement présentation tient compte de tous les objets présents lors de l'édition et du runtime.

Contrôle automatique du mouvement: le mouvement attend la pression sur les flèches et change l'index comme indiqué.
Changement automatique de scène: le mouvement change la scène lorsque le dernier index a été traité.
Termine automatiquement l'index précédent: cochez cette case pour forcer la terminaison de l'index précédent avant de gérer l'index suivant.
