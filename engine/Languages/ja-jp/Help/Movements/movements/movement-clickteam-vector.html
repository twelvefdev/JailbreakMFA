ベクター

概要

一つの動作とコントローラーエクステンションで非常に多くの種類の動作を作成します。 この動作は、加速と速度を格納し (それらは毎ループ更新されます)、シンプルにオブジェクトをコントロールします。 この動作は、 Clickteam 動作コントローラー エクステンションが提供する多彩な操作オプションと組み合わせて使用することによって威力を発揮します。 詳しくはサンプルファイルをご確認ください。 注意: MMF1.5 ユーザーの方には、この動作が "Movement Extension" の代替になります。 CF 2.5 には "Movement Extension" は含まれていません。

動作の使用

ベクター動作は非常にたくさんの方法で使用が可能ですが、ほとんどの場合、コントローラーエクステンションが必要で、フレームに追加しておく必要があります。 プロパティの初期値は必要に応じて変更してください。 ただし、コントローラー エクステンションとイベントエディタを使用することでランタイム時に変更できますので、初期値はあまり重要でない場合もあります。 この動作の詳細は付属のサンプルファイルで確認することを強くお勧めいたします。 重力/加速が不要の場合は、強さを 0 に設定してください。

フレームエディタでのプロパティ

図 1: ベクター動作のプロパティ。

全般のプロパティ

開始時に移動: フレームの開始時に動作を開始するかを決定します。
アニメーションの方向を自動的に変更: このオプションが有効の場合、動作の方向をベースにしてアニメーションの方向を自動的に変更します。 場合によっては手動で方向を変更したいこともあるでしょう。

初期方向と速度

速度: オブジェクトの初期速度です。オブジェクトは、毎フレームここで設定した値 x 0.1 で移動します (加速しない場合)
方向: オブジェクトが移動する初期方向 (角度) です。

重力 / 加速

強さ : 重力/加速の強さです。 オブジェクトの速度は、マイフレームここで設定した値 x 0.1 の係数で変化します。
方向: オブジェクトに働く重力/加速の方向 (角度) です。
